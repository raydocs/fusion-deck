# Ultra two-round workflow

`ultra_two_round` raises the ceiling by spending the second round on uncertainty, not repetition.

1. Build a compact context/evidence packet.
2. Run a first blind panel with the same prompt for every panelist.
3. Judge consensus, contradictions, blind spots, and uncertainty.
4. Assign targeted probes only for blocking claims.
5. Run deterministic verifiers where possible.
6. Final judge synthesis: decision, verified evidence, rejected alternatives, residual uncertainty.

First-round answers should include:

- answer / recommendation;
- assumptions;
- evidence used;
- likely failure modes;
- what would change my mind.

The final answer must disclose realized panel state/mode, targeted probes, verifier results, and run id.
